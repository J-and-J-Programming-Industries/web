About | Download Emulator | Install | Links

Installing JOS is easy! You don't even need to wipe your hard drive! You can just download the Windows version here as an executable, or the Linux version at the links, then running them!
After you have downloaded them, double click on the .exe file on Windows, or type ./JOS in the terminal on Linux.
It is a command line based program, so your desktop will continue to run in the background.
All programs you have already will still work, as it does not replace your native OS.
Soon, a downloadable OS .iso will be available that will contain JOS and some programs with it, alongside a Arch Linux backbone for running programs not compatible with JOS.
